API
Application Programming Interface
=> 한 프로그램에서 다른 프로그램으로 데이터를 주고받기 위한 방법

ex) 유저와 서버가 데이터를 주고받는 방법(=코드)
API는 요청방식(보낼 것인지 받을 것인지),
자료 요청(뉴스, 웹툰 등), 
파라미터(세부 정보)를 가져가야한다.
+ 웹의 경우 REST API 원칙에 따라 작성하면 좋다.

유저도 GET 요청을 하는 코드를 짜야한다.
=> 웹브라우저의 주소줄 or 구현된 버튼, 그림 등에 API 요청 코드가 있다.

public API : 누구나 사용가능한 공개 API
private API : 사내에서만 공유하는 API
Partner API : 특정 대상만 사용하는 API

모든 프로그램은 API를 가질 수 있음.
Windows API : 윈도우 운영체제 기능들을 사용가능.
Datavase 관리프로그램 API : DB 입출력 기능들을 사용가능.
~~~ 프로그램 API : ~~~ 기능들을 사용가능.
API로 만드는 비즈니스 모델 : 개쩌는거 만들면 판매도 가능.
