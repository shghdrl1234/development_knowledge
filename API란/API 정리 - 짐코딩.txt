API
Application programming Interface
응용프로그램에서 사용할 수 있도록, 운영체제나 프로그래밍 언어가 제공하는 기능을
제어할 수 있게 만든 인터페이스.

Interface란 ?
상호간에 소통을 위한 접점.

ex) 사람 - 스마트키 - 자동차
사람이 자동차 문을 열기위한 인터페이스는 스마트키가 된다.

그렇다면 API는 응용프로그램에서 소통을 위한 접점이다.
응용프로그램은 운영체제(안드로이드, 윈도우 등)에서 실행되는 모든 소프트웨어다.

즉, API는 애플리케이션에서 데이터를 읽거나 쓰기 위해 사용하는 인터페이스다.

대부분 업무에서 API라고 하면 HTTP API를 지칭한다.

private API => 사내에서만 사용가능한 API

public API => 오픈 API
= > 대부분은 HTTP API이며, 데이터를 주고 받을 때 사용하는 
프로토콜 = 소통방법 = 통신방법 = 통신규약으로 통한다.
